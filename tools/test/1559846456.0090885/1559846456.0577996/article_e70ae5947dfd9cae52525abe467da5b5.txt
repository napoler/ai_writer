> 什么是腹泻？
>>腹泻是一种常见的症状，俗称“拉肚子”，是指排便次数明显超过平日的正常频率，粪质稀薄，水分增加，或者含有未经消化的食物、脓血、黏液等。
也可以理解为消化物快速通过肠道，导致以下一种或多种情况：肠道蠕动频率增加、粪便稀或粪便量增加。

 > 如何对腹泻的病因做出诊断？
 >>> 导致腹泻的原因有很多。
当狗狗出现腹泻时，找到造成腹泻的原因是很重要的， **只有正确的找到原因才能够对症下药、有效治疗**。
兽医会结合你提供的信息以及体检和实验室诊断结果来确定腹泻的病因。
>> >>> **1、确定病灶：（大肠or小肠）**>>有很多的原因可以导致腹泻，所以首先我们要确定是肠道的哪一部分出现了问题才导致的腹泻。
通过定位是小肠还是大肠的原因导致了腹泻，能够帮助我们将病因的范围缩小。
通过下表您能够比较简单容易地确定病灶的位置是在大肠还是小肠。
>> > 注：【努责】医学上指大便或分娩时腹部用力>> >>> **2、症状发作的情况：**>> 症状出现的有多突然也是用来判断腹泻原因的一个很重要的线索。
如果症状突然出现，这种情况称为 **急性腹泻**。
如果症状出现并持续了一个很长的时间（数周），这种腹泻被称为 **慢性腹泻** ；如果症状出现，消失，然后又重新出现，持续数周，那么这种腹泻就被称作**间歇性腹泻** 。
>> >>> **3、病史**>> 狗狗的病史对腹泻的诊断也很重要，这包括 **疫苗免疫** 的情况、使用的 **驱虫药种类** 及 **驱虫频率** ， **与其他狗狗接触** 的情况，**饮食情况** ，是否存在有 **吃下垃圾或者毒物** 的可能，以及 **用药情况** 。
你提供的信息越多，越有利于医生做出正确的判断。
>> >>> **4、身体检查**>> 兽医需要对你的狗狗进行一系列完整的身体检查，包括 **测量体温** 、 **检查心脏和呼吸** ，观察 **口腔** 情况，进行 **腹部触诊**，检查是否存在 **脱水** 以及进行 **直肠检查** 。
>> >>> **5、实验室诊断**>> **几乎所有的腹泻的病例，兽医都需要进行粪** **检。
**由此来判断是否存在寄生虫的感染，如蛔虫或贾第鞭毛虫。
如果怀疑有细菌感染，那么就需要做粪便细菌培养和药敏试验。
如果是一些特定的病毒性疾病如细小病毒感染，还有其它的粪便检查能够帮助诊断。
>> **如果你的狗出现生病的迹象** ，那么经常会推荐全血细胞计数和血液生化检查。
如果怀疑某个特定的疾病也可以进行特定的血液检查。
>> **如果被怀疑存在肿瘤、异物或解剖结构问题**，还需要进行X线片检查。
其他一些影像学的手段如钡餐或超声有些时候也需要用到。
当然，内窥镜或结肠镜在某些情况下也会用到。

 > 腹泻如何治疗 >>> 因为造成腹泻的原因多种多样，那么治疗的方法也就不一而足了。
（见下图犬腹泻病因，诊断与治疗）>> >>> 在出现腹泻症状时，很多情况下先进行一个**24小时的禁食，在这段时间里只喂少量的水（可多次频繁给水）。
然后再给一些清淡的食物（如少量的煮熟汉堡或米饭等）。
如果此时腹泻不再复发，那么就在下面的几天时间里慢慢的恢复狗狗的正常饮食或特定饮食。
**>> >>> 而对于有些狗狗的腹泻，可能需要永久的 **调整狗狗的饮食**。
这时候可能需要一些特殊的食物去避免某些物质的摄入，比如在饮食中增加纤维、减少脂肪摄入、增加食物转化率等。
>> >>> **如果存在肠道寄生虫，应给予适当的驱虫药。

对于肠道寄生虫的驱虫，应该引起我们注意的不仅仅是狗狗体内的寄生虫，还有环境中的虫卵，消除狗狗生活的环境中的虫卵也非常的重要，这需要我们对狗狗生存的环境进行定期有效的清理以及消毒。
我们文章前面提到的粪检中用到的方法可以检测到粪便中的虫卵，如果粪便中没有虫卵但是有寄生虫的成虫或幼虫，那么结果也是阴性的。
所以，即便粪检的结果是阴性的，这也不能排除狗狗的肠道中没有寄生虫存在，依然还是需要驱虫的。
>> >>> **如果出现了脱水症状，需要及时对狗狗采取静脉或皮下输液补水。
**而口腔补液很多时候并不适用，因为液体很快穿过动物消化道排除体外，以至于动物不能够充分吸收水分。
>> >>> **如果腹泻是因细菌引起的，那么必须使用抗生素了**。
当出现肠道损伤的时候，也是需要使用抗生素的（当粪便中出现血液时即提示肠道有损伤）。
损伤的肠道给了肠道中的细菌进入血液的一个机会，如果不及时采取抗生素治疗可能就会导致更严重的后果（败血症）。
>> >>>在某些病例中，一些抑制肠道蠕动的药物会被用到，通过降低肠道的蠕动来降低食物的通过速度，从来提高吸收率。
而这些药物是不能用来治疗因中毒或者细菌感染引起的腹泻的，所以，在使用药物前做出准确的诊断是非常重要的。
